microzord has a great semantic integration with Angular framework.

Package installation

Install the core package if you do not have it yet:

npm i @microzord/core

Install Angular package for ideomatic working with Angular (supported versions are 11 and later):

npm i @microzord/angular

What is next

You can use microzord with Angular both ways:

  • As a host to insert other application in Angular app
  • As a child to prepare application to be inserted in another app
